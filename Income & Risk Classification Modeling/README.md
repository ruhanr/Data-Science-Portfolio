# Income & Risk Classification Modeling

## Overview
This project evaluates multiple classification models to predict income level and credit risk outcomes using structured tabular data. The focus was on systematic model comparison, validation, and performance optimization.

A detailed explanation of research design and findings is available in the accompanying Project Report.

## Data Sources
- Public income dataset (UCI Adult dataset)  
- Credit risk classification dataset  

## Methods
- Logistic Regression  
- Linear Discriminant Analysis (LDA)  
- Support Vector Machines (Linear & RBF)  
- Decision Trees with depth tuning  
- Cross-validation for model comparison  

## Results
- 83.69% accuracy with optimized Decision Tree model  
- 77.9% accuracy in credit default prediction using Logistic Regression  
- Implemented structured model evaluation workflow to mitigate overfitting
